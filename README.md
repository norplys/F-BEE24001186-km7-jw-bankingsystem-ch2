## Getting Started

How to run this project
```bash
npm i

edit your DATABASE_URL on .env file

npm run db:migrate
#or
npm run db:reset

npm run start
#or
npm start
```

Note: to run the sql you need to enstablish connection to your databse first and input it manually ( for now )
